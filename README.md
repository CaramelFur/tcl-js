# tcl-js
A port of tcl.js to NodeJs, a native javascript tcl interpreter

Currently working tcl commands:
* append
* apply
* array
* break
* catch
* clock
* concat
* continue
* dict
* error
* eval
* expr
* for
* foreach
* if
* incr
* index
* join
* lappend
* lassing
* list
* llength
* lmap
* lrange
* lrepeat
* lreverse
* proc
* return
* set
* split
* string
* switch
* throw
* try
* unset
* while
